## 🔍 عن المشروع
تم تصميم هذا المشروع لإدارة معهد تعليمي وتسهيل تنظيم البيانات المتعلقة بالطلاب، المدرسين، الكورسات المتاحة، الأقسام، والدبلومات. يتميز المشروع بواجهة مستخدم سهلة مقسمة عبر Master Page موحدة لتسهيل التنقل والتصفح.

---

## ✨ الميزات الأساسية
- **نظام تسجيل الدخول (User Login):** واجهة مخصصة لتأمين الوصول للوحة التحكم.
- **إدارة الطلاب (Student Management):** إضافة طلاب جدد بالبيانات الشخصية والدراسية وعرض بياناتهم.
- **إدارة المدرسين/المدربين (Trainer Management):** إضافة وإدارة بيانات الكادر التعليمي.
- **إدارة الكورسات (Courses Management):** إدارة وتحديث الدورات التدريبية، مدتها، تكلفتها، وتاريخ بدايتها.
- **إدارة الأقسام (Department Management):** تنظيم المعهد إلى أقسام متعددة لتسهيل الفرز.
- **إدارة الدبلومات (Diploma Management):** ربط الكورسات بدبلومات معينة لتنظيم الخطط الدراسية.
- **عرض التقارير والجداول (Display Pages):** صفحات خاصة لعرض قوائم الطلاب، المدرسين، الكورسات، الأقسام، والدبلومات بشكل مباشر وجداول منظمة.

---

## 🛠️ التقنيات المستخدمة
- **بيئة العمل:** .NET Framework 4.7.2
- **لغة البرمجة:** #C
- **تصميم الواجهات:** ASP.NET Web Forms (`.aspx`), HTML5, CSS3, JavaScript
- **قاعدة البيانات:** SQL Server LocalDB (ملف `Institute.mdf` المرفق داخل مجلد `App_Data`)
- **الاتصال بقاعدة البيانات:** ADO.NET (عبر كلاس `DB.cs`)

---

## 📁 هيكل المجلدات
* **`Institute Project/`**: المجلد الرئيسي للمشروع ويحتوي على:
  * **`App_Data/`**: يحتوي على قاعدة البيانات المحلية `Institute.mdf` وملف السجلات `Institute_log.ldf`.
  * **`My Page/`**: يحتوي على جميع صفحات الإدارة والعرض (إدارة الطلاب، الكورسات، المدربين...) بالإضافة إلى Master Page (`MyMaserPage.Master`).
  * **`Css/` & `JavaScript/`**: ملفات التنسيق البرمجية الخاصة بالواجهات.
  * **`DB.cs`**: الكلاس المسؤول عن فتح الاتصال وتنفيذ الاستعلامات وجلب البيانات من قاعدة البيانات.
  * **`User Login.aspx`**: صفحة تسجيل الدخول الرئيسية للمشروع.
  * **`Web.config`**: ملف إعدادات المشروع وسلسلة الاتصال بقاعدة البيانات.
* **`packages/`**: مجلد حزم NuGet (تم استثناؤه من الرفع عبر Git لسهولة التحميل التلقائي).
* **`Institute Project.sln`**: ملف الحل الرئيسي لفتح المشروع في Visual Studio.

## 🚀 طريقة التشغيل والتهيئة

### 1. المتطلبات الأساسية:
- بيئة **Visual Studio 2022** أو أحدث.
- تثبيت حزمة **ASP.NET and web development** أثناء تثبيت Visual Studio.
- تثبيت **SQL Server Express LocalDB** (يأتي افتراضياً مع حزمة أدوات البيانات في Visual Studio).

### 2. خطوات التشغيل:
1. قم بتحميل أو عمل `Clone` للمستودع من GitHub.
2. افتح ملف الحل `Institute Project.sln` باستخدام Visual Studio.
3. سيقوم Visual Studio تلقائياً باستعادة الحزم (NuGet Packages Restore).
4. تأكد من أن سلسلة الاتصال في كلاس `DB.cs` مهيأة بشكل صحيح للعمل بشكل ديناميكي:
   ```csharp
   private string ConnectionString = @"Data Source=(LocalDB)\MSSQLLocalDB;AttachDbFilename=|DataDirectory|\Institute.mdf;Integrated Security=True";
   ```
5. اضغط على مفتاح `F5` أو انقر على **IIS Express** لتشغيل المشروع في المتصفح.
